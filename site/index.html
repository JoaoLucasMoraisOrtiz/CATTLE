Built for context precision

Every feature exists to give your agents exactly the right context — nothing more, nothing less.

📚

Knowledge Base + RAG

Add docs, specs, and architecture files. Hybrid search (semantic embeddings + BM25) finds the right chunks automatically. Injected into agent context on every message.

📝

Prompt Builder

Type your intent → ReDo! finds relevant code via AST + embeddings → you select what to include → rich prompt with file references. The agent reads files with its own tools — zero wasted tokens.

🔗

AST Code Graph

Tree-sitter parses your code into a navigable D3.js graph. See call relationships, expand connections, right-click to view code or ask an agent to explain.

⏱

Temporal Decay Search

When searching agent logs for a commit, messages from the same day rank 10× higher than messages from last month. Because timing matters.

🤝

Cross-Agent Intelligence

What agent A learned is searchable by agent B. Human selections in the Prompt Builder become knowledge for future searches.

🖥

Real Terminal Multiplexer

Each agent runs in a real PTY with xterm.js. Kiro, Gemini, Claude, Codex — all with native terminal I/O, not a chat wrapper.

📊

Git Log + Diff Viewer

VS Code-style diff viewer with commit timeline. Multi-repo support, branch filtering, local commit detection. Explain diffs from agent logs or ask an agent.

🔧

MCP Server Per Agent

Each agent gets a ReDo MCP server with tools: save/update task analysis, search knowledge base, list analyses. Agents document their work for others.
